# Live Rundown

## Purpose

Digital replacement of the traditional wedding Excel rundown.

Designed for real-time wedding operations.

---

## Core Fields

- Time
- Program
- Used Time
- Food Serving
- Song
- Screen
- Remarks
- Responsible Roles
- Coordinator
- Status
- Sequence

---

## Viewing Modes

### Full Rundown

Default view.

Every wedding member can view the complete rundown.

---

### My Rundown

Optional filter.

Shows only activities matching the selected Responsible Roles.

This does NOT restrict access to the Full Rundown.

---

## Editing

Editable by

- Planner
- Coordinator (if permission granted)
